# Classwork
Practice of using 2d array

Objectives: 

Write a function which takes an input of an 2d array and ouput the array in echelon form. The size of the array is 3x4.

For example:

Input:
```
1 -1 -1 3 
1 1 -2 1 
4 -2 4 1 
```
Output:
```
1 -1 -1 3 
0 1 -0.5 -1 
0 0 1 -1 
```
